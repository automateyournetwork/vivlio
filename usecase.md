vivlio - Transform Meraki REST APIs into Business Ready Documents
=====================================

vivlio uses various Meraki REST APIs and transforms them, using Jinja2 templates into:

RAW JSON files

RAW YAML files

CSV files

Markdown files

Dynamic HTML pages

Mindmaps
